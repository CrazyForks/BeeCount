BeeCount 商店截图生成

点导出后会弹"选文件夹"对话框（Chrome/Edge），选 assets/store/screenshots/output/，所有图直接写进去，无需手动 mv。
Safari / 旧浏览器会 fallback 成 zip 下载，自己解压到 output/。
必须本地 http server 运行： cd /path/to/BeeCount && python3 -m http.server 8000 --bind 127.0.0.1 → 打开 http://localhost:8000/assets/store/screenshots/templates/index.html。 （file:// 浏览器拒绝读本地图 + 阻止 FS API。）
